# fuzzing-labs
Реализован мутационный фаззер для бинарного файла vuln4.exe.

Возможности фаззера:

Замена одного байта,
Замена нескольких байт,
Дозапись данных в конец файла,
Подстановка граничных значений (0x00, 0xFF, 0xFFFFFFFF и др.),
Последовательный режим мутации (sequential mode),
Мутации рядом с разделителями ,:=;,
Расширение ASCII-полей,
Сбор покрытия кода через DynamoRIO (drcov),
Автоматическое обнаружение crash,
Сохранение входных данных, вызвавших сбой,
Интеграция с DrMemory для анализа памяти.
